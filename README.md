## Howdy
videos sharing app created with react and sanity

## Technologies used:
- Nextjs
- Sanity
- Tailwindcss
- Zustand

## Features
- GoogleLogin
- Like Video
- Search Video
- Filter Videos (based on topics)

## How to run

First, install dependencies:
```bash
npm install
# or
yarn 
``````


Then, start the local development server http://localhost:3000:
```bash
npm run dev
# or
yarn dev
``````
